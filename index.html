Quick Start

Comandos para rodar o projeto

📄 Processar um PDF
Extraia texto de um PDF individual para TXT.
$ python src/pdf_to_txt.py --input data/raw_pdfs/nome.pdf --output data/txt
📦 Processar em Lote
Processe todos os PDFs da pasta de uma vez.
$ python src/batch_processor.py
🤖 Gerar CSV com IA
Converta os TXTs em CSV estruturado usando OpenAI.
$ python main.py
⚙️ Instalar Dependências
Configure o ambiente com todas as libs necessárias.
$ pip install -r requirements.txt
